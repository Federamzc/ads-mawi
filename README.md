# MAWI Consulting

A modern, dark-themed landing page for MAWI Consulting — strategic growth consulting for small businesses.

## Tech Stack

- **Next.js 15** with App Router
- **TypeScript**
- **Tailwind CSS 4**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```
